# Corporate Site Template

Bootstrap 5を使用した、モダンなコーポレートサイトのテンプレートです。

## 特徴
- **Bootstrap 5 採用**: 最新のCSSフレームワークによるレスポンシブデザイン。
- **グラデーション背景**: 全体的、およびヒーローセクションに美しいグラデーションを適用。
- **カードレイアウト**: サービス紹介等に最適な、視認性の高いカードデザイン。
- **ハンバーガーメニュー**: スマートフォン等のモバイル端末に完全対応。
- **お問い合わせフォーム**: お名前、メール、メッセージの入力が可能なフォーム。
- **SNSリンク**: フッターに主要なSNSへのリンクアイコンを配置。

## 使用技術
- HTML5
- CSS3 (Bootstrap 5)
- JavaScript (Bootstrap 5 Bundle)
- Bootstrap Icons

## セットアップ
`index.html` をブラウザで開くだけで確認できます。
